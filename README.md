# Social Media Management & Design Internship 🚀
Organization: Internee.pk
Role: Social Media Management Intern
Creator: Zarkish (BS E-commerce Student at Punjab University)

 Project Overview
This repository showcases a complete social media marketing suite developed for Internee.pk. 
The goal was to create "scroll-stopping" visuals while maintaining a highly organized back-end system for content distribution.

 Skills & Tools Used
Design: Canva (Typography, Branding, Visual Hierarchy)

Planning: Google Sheets (Content Scheduling, Status Tracking)

Management: GitHub (Documentation & Version Control)

Strategy: Digital Marketing & E-commerce (Target Audience Analysis)

 What's Included in this Repository
1. Content Calendar & Strategy 
A functional tracker built in Google Sheets to manage the content lifecycle.

Functionality: Includes custom dropdown menus for post status and category tracking.

Logic: Structured to balance educational, hiring, and engagement-based content.

2. Social Media Visual Assets 
2 Single Image Posts: Targeted at students with "Zero Experience," highlighting the accessibility of 15+ domains.

High-Impact Reel Cover: Designed with high contrast to drive engagement and highlight the "Work Remote" benefit.

Carousel Strategy: A 3-slide PDF guide designed to simplify the user enrollment process for new interns.

 Design Strategy & Execution
To showcase my talent as a Social Media Manager, I applied the following principles:

Brand Consistency: Strictly followed the Internee.pk color palette (Green/White/Black) to build trust.

User Focus: Simplified complex information into digestible "quick-read" formats.

Action-Oriented: Every asset includes a clear Call-to-Action (CTA) to drive enrollment.
